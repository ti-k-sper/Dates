## Exercice:  

A l'aides d'un algorythm sur javascript, ainsi que l'utilisation de html et css, nous devons créer une page qui nous indiquera, une fois notre date de naissance envoyée, notre age sur :  
- **Terre**
- **Vénus**
- **Mercure**
- **Jupiter**
- **Mars**
- **Saturne**
- **Pluton (rip)**
